DTO (데이터 전송 객체 : Data Transfer Object)
  : 데이터를 저장하거나 전송하는데 사용되는 객체
    다른 코드 없이 오로지 데이터를 담고있는데만 사용
    데이터만 가지고 있는 객체 -> 값 객체(= Value Object = VO)

DTO를 작성하기 위해서는 자바빈즈 규약에 따라서 작성
 1. 자바빈즈는 기본(default)패키지 외의 패키지에 속해야 함.
 2. 멤버 변수의 접근제어자는 private로 선언해야 함.
 3. 기본생성자가 존재해야 함.
 4. 멤버 변수에 접근할 수 있도록 getter, setter가 있어야 함.
 5. getter, setter의 접근 제어자는 public으로 선언해야 함.
