인파라미터(in parameter) : 미리 작성해둔 쿼리문에서 일부 값을 나중에 결정할 수 있게 해주는 매개변수. '?'로 표시

"INSERT INTO MEMBER VALUES(?, ?, ?, ?)"

DB 사용 작업 순서
 1. DB 연결
 2. 입력값 준비
 3. 쿼리문 생성
 4. 쿼리문 실행
 5. DB 연결 닫기

DB연동
 - DTO Class : 데이터를 교환하기 위해 생성하는 객체.
   (VO)	     : 일반적으로 테이블당 하나씩 생성.
	     : 테이블의 컬럼과 동일한 멤버변수를 가짐.
	     : 다른 코드 없이 속성(멤버변수)과 속성에 접근하기위한 게터/세터만 가짐
 - DAO Class : 데이터베이스의 데이터에 접근하기위한 객체. 보통 JDBC로 구현하고 CRUD를 담당.
